# AI CAD

AIファーストな建築CADアプリケーション。誰でも無料で使えるオープンソースCAD。

## 特徴

- **完全無料** - ローカルLLM（Ollama）で完全無料運用
- **AI連携** - 自然言語で図面を操作・修正
- **クロスプラットフォーム** - Windows / Mac / Linux 対応
- **JWW / DXF対応** - 既存図面をそのまま読み込める
- **オープンソース** - MIT License

## 使い方

```bash
npm install
npm start
```
